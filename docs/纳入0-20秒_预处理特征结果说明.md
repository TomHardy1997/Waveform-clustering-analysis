# 纳入 0-20 秒 Excel 批量预处理和特征提取结果

## 输入

输入数据来自：

`/Users/tangdi/Desktop/波形Excel数据/整理后_0-20秒原始Excel/纳入_0-20秒Excel`

处理文件数：`165` 个。

## 处理内容

每个文件按照示例 `处理结果_113520-2.xlsx` 的形式生成处理结果，包括：

- 原始数据
- 预处理后数据
- 峰值特征
- 波谷特征
- 分阶段特征
- 整体统计特征

## 时间段定义

- 刺激前：0-5 秒
- 刺激期：5-10 秒
- 撤光后：10-20 秒

## 输出

- `单文件独立处理结果`：每个细胞一个 Excel
- `单文件预处理波形图`：每个细胞一张预处理和峰谷标记图
- `总览图`：全部波形叠加、热图、特征分布
- `纳入0-20秒_批量预处理特征汇总.xlsx`：总特征表
- `纳入0-20秒_预处理特征可视化.ipynb`：查看结果的 notebook

## 说明

这版处理参考前人脚本的预处理步骤：异常值截断、基线校正、Savitzky-Golay 平滑、峰值检测。额外增加了波谷检测、分阶段特征、ON/OFF 特征和用于聚类的核心特征。
